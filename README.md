# 小願望達摩帳

用達摩記錄小願望的靜態 PWA 網頁。

## 功能

- 新增、完成、刪除願望
- 未達成區 / 已達成區
- 填寫願望後點左眼，完成後點右眼
- Nippon Colors 隨機達摩色票
- PWA manifest 與 service worker
- Firebase Email/Password 登入與 Firestore 同步設定

## 本機啟動

在 `Daruma` 資料夾執行：

```powershell
.\start-daruma-local.ps1
```

然後開啟：

```text
http://localhost:8765/daruma-wishes.html
```
